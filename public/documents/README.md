# ドキュメントディレクトリ

このディレクトリには、ダウンロード可能なPDFやPPTXファイルを配置します。

## 使い方

1. PDFまたはPPTXファイルをこのディレクトリに配置します
2. ファイル名を適切に設定します（例: `sample.pdf`, `presentation.pptx`）
3. コンテンツページから `/documents/ファイル名` としてリンクを設定します

## 例

- `static/documents/sample.pdf` → サイト上では `/documents/sample.pdf` としてアクセス可能
- `static/documents/presentation.pptx` → サイト上では `/documents/presentation.pptx` としてアクセス可能

## 注意事項

- ファイルサイズが大きい場合は、読み込み時間に注意してください
- 機密情報を含むファイルは配置しないでください
- ファイル名は英数字とハイフン、アンダースコアのみを使用することを推奨します
