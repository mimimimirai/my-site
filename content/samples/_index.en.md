## How to Upload Files

1. Place PDF or PPTX files in the `static/documents/` directory
2. Set the file name appropriately
3. You can download from the links on this page

**Note**: After placing files, please rebuild the Hugo site.

## About Security

- All files are provided as static files
- Files are provided as direct download links and can be opened in a browser
- You can change the file name to restrict access if necessary
